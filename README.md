## Getting Started


### Prerequisites

* [Raspberry Pi 3](https://https://www.raspberrypi.org/products/raspberry-pi-3-model-b/)
* ALSA
* PulseAudio
* Python >= 3.6.0
* USB Microphone
* Tkinter
